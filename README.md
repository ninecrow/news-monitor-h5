## 📌 项目简介

专为股票投资者打造的实时突发新闻监控页面，聚合多平台财经资讯，每条新闻自动匹配**应对SOP（标准操作程序）**，帮助投资者在突发消息面前快速做出理性决策。

**核心能力：**
- ⚡ **实时聚合** - 30秒自动刷新，多源同步
- 🎯 **消息分级** - 紧急/警告/一般三级分类
- 📋 **智能SOP** - 每条消息附带应对策略
- 📱 **响应式设计** - 支持PC/手机/平板

## 📋 SOP应对策略

每条新闻底部自动附带5步应对策略：

### 示例：政策类消息SOP
```
📋 【股神】应对SOP
1. 立即查看持仓股是否受利好（金融/地产板块）
2. 如高开>2%，暂停新买入计划
3. 已持仓且盈利的，可考虑减仓10-20%
4. 关注成交量是否放大验证利好
5. 尾盘决定是否加仓，不追高
```

### 示例：风险类消息SOP
```
📋 【股神】应对SOP
1. 立即减仓50%，锁定部分利润
2. 观察市场情绪，如跌超-3%清仓
3. CGD评分下调，暂停加仓计划
4. 关注后续是否有机构接盘
5. 3日内不反弹，全部离场
```

## 🔧 高级配置

### 修改数据源
```javascript
// index.html 中修改 CONFIG
const CONFIG = {
    CURRENT_SOURCE: 'caixun',  // 'caixun' 或 'juhe'
    REFRESH_INTERVAL: 30,       // 刷新间隔(秒)
    JUHE_API_KEY: 'your_key'    // 聚合数据KEY
};
```

### 切换数据源
点击页面上的 **📡 切换数据源** 按钮，或在控制台执行:
```javascript
CONFIG.CURRENT_SOURCE = 'juhe';  // 切换到聚合数据
refreshNow();  // 立即刷新
```
